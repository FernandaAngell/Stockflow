# 📦 StockFlow

## 📚 Descripción

Este repositorio contiene el desarrollo de un sistema de inventario web 💻 orientado a la gestión de productos, control de stock y registro de ventas.

El proyecto fue desarrollado como parte de mi aprendizaje en desarrollo Full Stack 🚀 utilizando tecnologías frontend, backend y base de datos.

Se trabajan conceptos fundamentales como:

- CRUD de productos 🛒
- Gestión de inventario 📦
- Registro de ventas 💰
- Conexión con base de datos MySQL 🗄️
- API REST con Node.js y Express ⚡
- Consumo de APIs con JavaScript
- Diseño responsive y dashboard administrativo 🎨
- Despliegue en Netlify y Render 🌐

---

# 📂 Contenido

## 📊 Dashboard
Visualización general del inventario, productos registrados, stock bajo y valor total del inventario.

## 📦 Productos
Gestión completa de productos:

- Agregar productos
- Editar productos
- Eliminar productos
- Control de stock

## 💰 Ventas
Registro de ventas y actualización automática del stock disponible.

## 📝 Historial
Visualización de movimientos y ventas realizadas dentro del sistema.
